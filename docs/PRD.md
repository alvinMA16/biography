# 回忆录产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位
一款面向老年人的回忆录生成工具，通过AI语音对话的方式，帮助老年人记录和整理人生故事，最终生成一份完整的文字回忆录。

### 1.2 核心价值
- 降低记录门槛：老年人无需打字，通过语音即可讲述故事
- 智能引导回忆：AI温和地引导老人回忆，帮助挖掘更多细节
- 自动整理成文：将零散的对话内容整理成连贯的回忆录文章

### 1.3 产品形态
微信小程序

---

## 二、目标用户

### 2.1 用户画像
- **年龄**：60岁以上的老年人
- **特点**：
  - 有丰富的人生经历想要记录
  - 不擅长或不方便打字
  - 会使用微信基本功能
  - 时间相对充裕
- **使用场景**：老人独立使用，不依赖家人协助

### 2.2 用户需求
- 想把人生故事留给后代
- 希望有人听自己讲故事
- 想整理自己的人生经历
- 害怕记忆随时间消逝

---

## 三、核心功能

### 3.1 语音对话系统

#### 3.1.1 语音输入
- 支持长按录音
- 语音转文字（ASR）
- 支持方言识别（可选，后续版本）

#### 3.1.2 AI回复
- 文字显示 + 语音播放
- **AI人设：温和的晚辈**
  - 像孙子/孙女一样，亲切地请教老人讲故事
  - 用尊敬的语气，称呼"您"
  - 表现出对老人故事的好奇和兴趣
  - 示例："爷爷/奶奶，您刚才说的那件事太有意思了，能再给我讲讲吗？"

#### 3.1.3 对话特点
- 每次对话5-10分钟（短对话模式）
- 积少成多，多次对话逐步完善回忆录
- 随时可以暂停，下次继续

### 3.2 智能追问系统

#### 3.2.1 追问策略
- **温和引导型**：像和蔼的晚辈在聊天
- 不催促、不打断
- 适时追问细节："那时候您几岁？""能给我讲讲当时的感受吗？"
- 关联追问："您刚才提到的那位朋友，后来怎么样了？"

#### 3.2.2 信息收集维度
- 时间：什么时候发生的
- 地点：在哪里发生的
- 人物：涉及哪些人
- 事件：具体发生了什么
- 感受：当时的心情和想法
- 影响：这件事对您有什么影响

### 3.3 回忆录生成

#### 3.3.1 内容范围（用户自选）
- 完整人生故事（从童年到现在）
- 特定主题（如工作生涯、家庭故事、难忘经历等）
- 自由讲述（想到什么讲什么）

#### 3.3.2 文字风格（用户可选）
- 第一人称叙述："我出生在..."
- 第三人称传记："他出生在..."

#### 3.3.3 生成机制
- 每次对话后自动整理该次对话内容
- 定期将多次对话内容整合成连贯的章节
- 支持按时间线或主题组织内容

### 3.4 内容管理

#### 3.4.1 查看回忆录
- 查看已生成的回忆录内容
- 按章节/主题浏览
- 支持文字朗读（TTS）

#### 3.4.2 编辑与修正
- 老人可以语音补充或修正内容
- AI理解修改意图并更新

#### 3.4.3 导出功能
- 导出为PDF文档
- 导出为Word文档
- 支持分享给家人

## 四、用户旅程

### 4.1 首次使用
1. 打开小程序，看到简洁温馨的欢迎页
2. 简单的功能介绍（图文引导，字体够大）
3. 选择想要记录的内容范围
4. 选择喜欢的文字风格
5. 开始第一次对话

### 4.2 日常使用
1. 打开小程序，看到上次对话摘要
2. 点击"继续聊天"或"开始新话题"
3. 进行5-10分钟的语音对话
4. 对话结束，AI总结本次内容
5. 查看/收听新生成的回忆录片段

### 4.3 查看成果
1. 进入"我的回忆录"
2. 浏览已整理的章节
3. 可以听AI朗读
4. 可以导出或分享

## 五、界面设计原则

### 5.1 适老化设计
- 字体大（至少16pt，关键信息更大）
- 按钮大、间距大
- 颜色对比度高
- 操作简单，层级少
- 避免复杂手势

### 5.2 界面结构
- 首页：开始对话 / 我的回忆录
- 对话页：语音输入区 + 对话历史
- 回忆录页：章节列表 + 内容详情
- 设置页：文字风格 / 导出 / 帮助

## 七、数据存储与隐私

### 7.1 存储方案
**用户可选**：让用户自己选择数据存储方式
- **云端存储**：存在服务器，支持多设备同步，需要账号登录
- **本地存储**：只存在用户手机，更私密，但换手机会丢失

### 7.2 隐私保护措施
- 数据加密传输和存储
- 明确的隐私政策说明
- 用户可随时删除自己的数据

## 九、其他设计决策

### 9.1 方言支持
MVP版本暂不支持方言，只支持普通话。后续版本可考虑接入方言识别能力。

### 9.2 敏感内容处理
**温和倾听**：不回避负面或创伤性回忆，这也是人生的一部分。AI应该：
- 耐心倾听，表达理解和共情
- 不急于转移话题
- 适当回应："这段经历一定很不容易""感谢您愿意分享这些"
- 如果老人情绪激动，温和询问是否需要暂停

### 9.3 分享方式
通过导出文档后分享：用户导出PDF/Word文档，然后通过微信发送给家人。

### 9.4 照片支持
MVP版本暂不支持照片，保持纯文字形式。后续版本可考虑图文结合。
